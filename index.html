Welcome to valyrian

Generate cool and unique names with javascript

How to use

var val = new Valyrian(2, 4);
          val.generateNames();

The generatesNames(sylMin, sylMax); function can accept 2 arguments :

  • sylMin: set the minimum syllables in your names (Default: 2)
  • sylMax: set the maximum syllables in your names (Default: 3)

You can change how many names you want by editing:

Valyrian.prototype.generateNames = function(nb) {
              var names = [],
              nb    = nb || 10; // Here

              for (var i = 0; i < nb; ++i) {
              names.push(this.createNewName(this.sylMin, this.sylMax));
              }
              this.names = names;
              return this.names;
          };
